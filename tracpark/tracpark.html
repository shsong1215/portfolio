PROJECT BACKGROUND

Tracpark

This application is for easier and convenient parking system. Busy people live with busy transportations. Since the world we live in is so fast paced and complex – we have a crazy parking system in big cities and it's a problem. Tracpark is both a mobile application and web application. The application focuses on swift communications with the user and the parking structure.

Tracpark will suggest a reservation system for parking, like how we make reservations to go to a restaurant.
It's as easy as that.
